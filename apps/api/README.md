# VoiceSense API

Public and internal API surface for AI employees, conversations, tools, workflows, and developer platform capabilities.

## Identity Foundation

The authentication and identity system lives in `app/identity` and includes users, organizations, memberships, teams, roles, invitations, sessions, OAuth accounts, verification tokens, and audit logs.

## Local Development

```bash
cd apps/api
python -m venv .venv
.venv\Scripts\activate
pip install -e .[dev]
uvicorn app.main:app --reload --port 8000
```

API docs are available at `http://localhost:8000/docs`.

## Migration

Run `python scripts/run_migrations.py` from `apps/api` to apply all pending PostgreSQL migrations before starting the API.

## Local Database Quick Start

See ../../infra/LOCAL_DEVELOPMENT.md for Docker Compose, migration, API startup, and identity smoke-test instructions.
