# Multi-Agent Routing Strategy

The Task 019 routing foundation selects target AI employees using role, team, availability, and workload placeholders.

## Routing Inputs

- Required role or expertise.
- Team membership.
- Availability state.
- Workload score.
- Permission and policy checks.
- Confidence score.
- Supervisor or department constraints.

## Future Improvements

Future routing should score expertise from agent configuration, tool permissions, historical performance, current workload, escalation risk, and customer context.
