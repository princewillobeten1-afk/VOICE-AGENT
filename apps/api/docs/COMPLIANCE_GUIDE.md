# Compliance Guide

Task 026 creates a compliance readiness system, not a certification claim.

## Supported Foundations

- Compliance framework records for SOC 2, ISO 27001, GDPR, HIPAA awareness, or custom programs.
- Evidence records linked to controls and optional storage files.
- Readiness scores and evidence summaries.
- Data governance policies for retention, residency, classification, and legal hold metadata.

## Operating Model

Compliance readiness should be reviewed by security, legal, and operations teams before being used in customer-facing claims.
