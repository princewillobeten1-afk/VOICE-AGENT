# Omnichannel API

## Catalog

- `GET /api/v1/omnichannel/channels/catalog`

## Channels

- `POST /api/v1/omnichannel/channels`
- `GET /api/v1/omnichannel/channels?workspace_id=...`

## Identity

- `POST /api/v1/omnichannel/identities/resolve`

## Sessions

- `POST /api/v1/omnichannel/sessions`
- `GET /api/v1/omnichannel/sessions?workspace_id=...`

## Messages

- `POST /api/v1/omnichannel/messages`
- `GET /api/v1/omnichannel/messages?workspace_id=...`

## Delivery

- `POST /api/v1/omnichannel/delivery-events`

## Timeline and Analytics

- `GET /api/v1/omnichannel/timeline?workspace_id=...`
- `GET /api/v1/omnichannel/analytics?workspace_id=...`
