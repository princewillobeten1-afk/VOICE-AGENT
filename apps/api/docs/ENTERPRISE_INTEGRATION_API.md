# Enterprise Integration API

## Marketplace

- `GET /api/v1/integrations/enterprise-marketplace`
- `GET /api/v1/integrations/marketplace`
- `GET /api/v1/integrations/available`

## Installation

- `POST /api/v1/integrations/installations`
- `GET /api/v1/integrations/installations?workspace_id=...`

## Sync

- `POST /api/v1/integrations/sync-jobs`
- `GET /api/v1/integrations/sync-jobs?workspace_id=...`

## Playground and Analytics

- `POST /api/v1/integrations/playground-runs`
- `GET /api/v1/integrations/analytics?workspace_id=...`

Existing connection, credential, action, trigger, health, log, API key, and webhook endpoints remain part of the same platform.
