# AIOps API

All endpoints are mounted under `/v1/analytics`.

- `GET /analytics/dashboard`
- `GET /analytics/live`
- `GET /analytics/breakdown`
- `POST /analytics/alerts`
- `GET /analytics/alerts`
- `POST /analytics/health`
- `GET /analytics/health`
- `POST /analytics/costs`
- `GET /analytics/costs`
- `POST /analytics/evaluations`
- `GET /analytics/evaluations`
- `GET /analytics/audit`
