# Enterprise Knowledge Management Architecture

Task 016A establishes the Knowledge Management Platform for VoiceSense. It is a governed content management system for AI employee knowledge, not the RAG engine.

## Core Domains

- Knowledge bases
- Documents
- Website registrations
- Connected sources
- Categories, folders, collections, tags, labels, metadata
- Versions and publishing
- Permissions
- Sync jobs
- Activity logs
- Quality checks

## Out of Scope

- Embeddings
- Vector databases
- Semantic search
- RAG retrieval
- AI reasoning
- Website crawling execution
