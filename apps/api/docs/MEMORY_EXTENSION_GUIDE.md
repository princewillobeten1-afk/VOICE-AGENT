# Memory Extension Guide

## Conversation Engine

Conversation turns should submit valuable user facts, preferences, and summaries to the Memory System after evaluation.

## Knowledge Engine

Knowledge should remain source-grounded. Memory should store learned context and structured facts, not replace the knowledge base.

## Workflow Engine

Working memory can track task state. Workflow execution remains separate.

## Tool Calling

Tools may propose memory updates, but policies and permissions decide whether to store them.

## Multi-Agent Collaboration

Shared memory allows multiple AI employees to coordinate through access-controlled memory records and links.
