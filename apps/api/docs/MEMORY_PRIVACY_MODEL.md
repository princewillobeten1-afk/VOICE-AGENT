# Memory Privacy Model

Memory privacy is critical because AI employees may learn sensitive customer and business facts.

## Controls

- Organization isolation
- Workspace isolation
- User ownership
- Agent ownership
- Visibility: private, workspace, organization, shared
- Privacy level: public, internal, restricted, private
- Memory access grants
- Audit logs and memory events
- Forget action for content removal

## Default Posture

The default privacy level is `internal`, and memories are scoped to workspace unless explicitly shared.
